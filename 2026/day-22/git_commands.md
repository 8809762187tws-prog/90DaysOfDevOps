GIT COMMANDS

1. git init - to convert a folder to git repo
2. git status - to check status
3. git add - to stage a file
4. git commit -m "" - to make a file tracked
5. git log - to check logs
6. git config --global user.name " " - to change author
7. git config --global user.email "  " - to change email
8. git branch - to check branch
9. git checkout -b - to create new branch
10.git checkout - to goto an existing branch
11.git switch - to switch branch
12.git branch -D - to delete a branch.
